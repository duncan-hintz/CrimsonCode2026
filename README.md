# CrimsonCode2026
Repository for Crimson Code 2026 Hackathon submission.

Poke-AI
This is a reinforcement learning algorithm to learn how to play Pokemon battles on the website Pokemon Showdown.
Currently this program simply learns and saves its learning, but there is a baseline here to develop continual learning and a bot that can fully play the game.

About the team:
This is a solo project from Duncan Hintz on the advanced track.
I am a senior computer science student at WSU.

HOW TO:
Clone the repo
Install npm/node in the workspace
Clone the pokemon-showdown repo
Run a local pokemon-showdown server with the command: npm pokemon-showdown start --no-security
Run the file CleanRLImplementation.py

The weights will be saved to models/testing/ for further implementation/learnning.
