Reflection Report
This was a submission for the Crimson Code Hackathon at WSU in Spring of 2026. The theme for the event was “reinventing the wheel”, creating something that iterates on an existing technology. 
The task was to create a piece of software that embodied the theme, while aiming to learn new skills and grow as a developer. I chose to make a project that improves upon the AI in Pokémon. 
I thought this would fit the theme as it takes an existing franchise of games and adds my take on reinforcement learning to improve upon it. 
It takes in a simulated battle, and outputs recommended moves based on a neural network algorithm. 

I made choices such as what information it should parse, what outcomes in a turn it should value (i.e. damage, status effects, KOs, etc.), and what system should be used to learn the game. 

I participated alone, but I am continuing this project with a small team of people. In this event specifically, I am happy with what I was able to accomplish as far as the AI learning outcomes, however I would be happier if I could have produced a product that could work out of the box as an application. 
I think if I were to redo this, or for the next time I enter a similar event, I would focus on completing a project that an end-user could use without needing to understand coding environments or other such ideas.
